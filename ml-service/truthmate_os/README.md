# TruthMate OS

A production-grade, multi-agent AI system for media forensics, link safety analysis, and fact consensus mapping.

## Setup

1.  **Dependencies:** `pip install -r requirements.txt`
2.  **Browsers:** `playwright install chromium`
3.  **Config:** Add your Gemini Key to `.env`
4.  **Run:** `python main.py`
5.  **Access:** `http://localhost:8000`
